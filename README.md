# Velocity Framework

A comprehensive TypeScript-first Node.js framework with decorators, built-in ORM, logging, and enterprise-grade features.

## Features

- 🎯 **Decorator-based** routing and middleware (@Get, @Post, @Middleware, @Interceptor)
- 🔄 **Advanced middleware** system with execution order control
- 🔀 **Request/response interceptors** for data transformation
- 🗄️ **Built-in ORM** with database abstraction (SQL and NoSQL support)
- 📝 **Structured logging** system with multiple output formats
- 💉 **Dependency injection** container for service management
- ✅ **Built-in validation** and serialization
- ⚙️ **Configuration management** with environment-based overrides
- 🔒 **Built-in security** features (CORS, rate limiting, helmet integration)
